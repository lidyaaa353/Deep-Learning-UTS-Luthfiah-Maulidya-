## ğŸ“ Ringkasan Tugas: Pengenalan Tensor (PyTorch)

Tugas dalam notebook ini mencakup pemahaman fundamental mengenai **Tensor**, yang merupakan struktur data utama dalam Deep Learning. Berikut adalah poin-poin yang dipelajari:

* **Inisialisasi Tensor:** Membuat tensor dari list, array NumPy, serta menggunakan fungsi `torch.ones()`, `torch.zeros()`, dan `torch.rand()`.
* **Atribut Tensor:** Mengecek `shape`, `dtype`, dan `device` (CPU/GPU) tempat tensor disimpan.
* **Operasi Tensor:** * Operasi aritmatika dasar (penjumlahan, perkalian).
    * **Matrix Multiplication:** Penggunaan `matmul` atau operator `@`.
    * **In-place Operations:** Memahami operasi yang langsung mengubah nilai variabel (ditandai dengan `_`).
* **Manipulasi Dimensi:** Teknik *indexing*, *slicing*, dan penggabungan tensor menggunakan `torch.cat`.
* **Bridge dengan NumPy:** Cara mengubah tensor menjadi array NumPy dan sebaliknya tanpa menyalin memori secara berlebihan.
